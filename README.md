# Exercícios da Faculdade

Repositório criado para armazenar os exercícios desenvolvidos durante a faculdade.

O objetivo principal deste projeto é praticar programação, organizar os códigos e também desenvolver familiaridade com o uso do Git e GitHub.

## Objetivos

- Praticar lógica de programação;
- Aplicar conceitos aprendidos em aula;
- Organizar exercícios e projetos acadêmicos;
- Praticar versionamento de código utilizando Git.

## Tecnologias

- Java
- Git
- GitHub

## Estrutura

Os exercícios estão organizados por disciplinas e conteúdos estudados durante o curso.
